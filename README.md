This is just a simple notes app i made in C++. remake of https://github.com/theKrillionaire/krillNotes


instructions:
  commands:
    make:
      make a note.
      use: `make <noteName> <noteContents>`
    read:
      print the contents of a note.
      use: `read <noteName> <noteContents>`
    show:
      show all notes.
      use: `show`
